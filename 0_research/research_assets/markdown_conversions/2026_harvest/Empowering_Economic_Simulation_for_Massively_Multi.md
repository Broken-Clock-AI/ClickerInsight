# Empowering Economic Simulation for Massively Multiplayer Online Games through Generative Agent-Based Modeling

## Abstract
Agent-Based Modeling (ABM) is a robust tool for analyzing game economics. This study introduces a novel approach using Large Language Models (LLMs) in MMO economy simulation. By leveraging LLMs’ role-playing proficiency, generative capacity, and reasoning aptitude, the authors design "MMOAgent" with human-like decision-making and adaptability.

## Key Components of MMOAgent
- **Profile:** Tailors the agent to mirror real player traits using data-driven design.
- **Perception:** Interprets the game environment through a semantic parser.
- **Reasoning:** Determines suitable actions using zero-shot chain-of-thought prompting.
- **Memory:** Utilizes both Short-Term Memory (STM) and Numeric-aware Long-Term Memory (LTM).
- **Action:** Executes structured actions with execution feedback.

## Experimental Results
- MMOAgent significantly outperforms baselines in capability and diversity metrics.
- The simulation replicates emergent phenomena like role specialization and market-aligned price fluctuations.
- LLM-empowered agents can promote price fluctuations in line with market rules.

## Conclusion
The integration of LLMs with ABM introduces "generative ABM," a new paradigm for studying MMO economies, enabling more realistic social interactions and direct player-to-player (P2P) trading simulation.

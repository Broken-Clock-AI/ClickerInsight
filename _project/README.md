# Idle Games Research Project

This repository contains a comprehensive research project on the topic of idle and incremental games. The project analyzes academic and design literature to produce a detailed tome on the genre's history, mechanics, and psychological impact.

## Project Structure

The project is organized into the following directories:

- **`/tome`**: Contains the final merged document, `Tome.md`, its prelude, and the individual chapter files (`partX_chapterY_title.md`). This is the primary output of the research project.

- **`/scripts`**: Houses the Python scripts used for various research and data management tasks, such as fetching research papers, generating lists, and extracting information.

- **`/research_assets`**: A directory for all the raw and processed research materials.
    - **`/downloads`**: Contains the original PDF and HTML files of the research papers and articles.
    - **`/markdown_conversions`**: Contains the markdown versions of the downloaded research materials, used for easier analysis.

- **`/analysis`**: Includes markdown files related to the research and analysis process. These files contain notes, plans, and outlines that informed the final tome.

## Usage

The primary content of this project can be found in the `/tome` directory. The `Tome.md` file is the complete, merged document, which can be read in its entirety. The individual chapter files are also available for more granular reading.

The scripts in the `/scripts` directory were used to support the research process. They are provided here for reference and can be adapted for similar research tasks.

The `.env` file in the root directory may contain environment variables used by the scripts. Ensure this file is properly configured if you intend to run the scripts.

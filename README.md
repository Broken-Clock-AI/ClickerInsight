# The Compounding Illusion: Research-Driven Incremental Design

> A multi-disciplinary exploration of idle mechanics, interpassivity, and the mathematics of growth.

## 🚀 Project Overview
This project is a deep-dive into the "Idle/Incremental" game genre. It bridges the gap between academic theory (psychology, economy, ludology) and practical implementation. 

**Our Goal:** To define and prototype a "hardened" incremental engine that utilizes formal game design principles to move beyond simple "number go up" mechanics into meaningful, research-backed player experiences.

---

## 🏗 Project Status: HFS v2.2 (Hardened Formal Specification)
We are currently in the **Execution Phase** of the High-Fidelity Synthesis (HFS) workflow.

- [x] **Phase 0: Research & Data Gathering** - Scraped 100+ game reviews, downloaded 20+ academic papers.
- [x] **Phase 1: Synthesis** - Authored "The Compounding Illusion" (v3.0), a 13-chapter research tome.
- [x] **Phase 2: Prototype v2.1 (RKA)** - Built a functional Vite/TypeScript engine using Recursive Knowledge Architecture.
- [ ] **Phase 3: Formal Hardening (v2.2)** - (Current) Transitioning to a formal YAML specification for zero-shot agent simulation and rigorous balancing.

## 📁 Repository Structure
- **`0_research/`**: Raw data, market analysis, and academic paper corpus.
- **`1_synthesis/`**: The "The Compounding Illusion" tome and analysis matrices.
- **`2_prototype/`**: The `mvp_vite_ts` game engine and design documents.
- **`scripts/`**: Python/PowerShell automation for research and balancing.

## 🛠 Tech Stack & Principles
- **Engine:** Vite + React + TypeScript + Zustand.
- **Math:** `break_infinity.js` for handling numbers up to $10^{308}$.
- **Design:** HFS (High-Fidelity Synthesis) v2.2 — Agent-ready formal specifications.
- **Philosophy:** "Aesthetics of delegated enjoyment" (Fizek, 2018) and "Calm Technology."

## 🗺 Roadmap
1. **Formal Simulation:** Building a headless simulation harness to verify 1000+ hours of game balance in seconds.
2. **Narrative Abstraction:** Implementing emergent storytelling systems based on *Universal Paperclips* and *Kittens Game*.
3. **Ethical Loops:** Designing monetization and retention strategies that respect player time and mental health.
